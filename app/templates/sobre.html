Sobre o PedagoType

O PedagoType foi desenvolvido como uma aplicação de treinamento de digitação com foco em textos pedagógicos. Utilizamos Flask no backend, MongoDB como banco de dados e uma API REST para o gerenciamento de usuários e pontuações.
