Sobre o PedagoType

O PedagoType é uma plataforma inovadora de treinamento de digitação com foco em conteúdos pedagógicos. Seu principal objetivo é permitir que os usuários aprimorem suas habilidades de digitação enquanto interagem com textos educativos.

A aplicação utiliza o framework Flask no backend, proporcionando uma estrutura robusta. MongoDB é o banco de dados escolhido, garantindo escalabilidade e gerenciamento eficiente das informações dos usuários.

Além disso, o sistema conta com uma API REST para gerenciamento de usuários e suas pontuações. A interface foi desenvolvida com HTML5, CSS3 e JavaScript, oferecendo uma experiência fluida em dispositivos móveis e desktops.

O PedagoType é uma solução eficiente para estudantes e profissionais que desejam melhorar suas habilidades de digitação e, ao mesmo tempo, aprender de maneira interativa e eficaz.
